# ANSI codes für farben

## Struktur
Startet immer mit `\033[` oder `\x1b[` <br>
z. B. "Hallo" Grün und Fett: `\033[1;32mHallo\033[0m`

## Farben
| Code | Farbe |
| :--- | :--- |
| 30m | Schwarz |
| 31m | Rot |
| 32m | Grün |
| 33m | Gelb |
| 34m | Blau |
| 35m | Magenta |
| 36m | Cyan |
| 37m | Weiß |

## Wichtige Steuerzeichen
| Code | Zeichen |
| :--- | :--- |
| 0m | Stile zurücksetzen |
| 1m | Fett / Hell |
| 2m | Dünn / Dunkel |
| 3m | Kursiv |
| 4m | Unterstreichen |
| 5m | Blinkend |
| 7m | Invertiert |
